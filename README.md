# React + Vite

A static webpage to act as a dashboard to track learner progress

Tech Stack  
-  React
-  Vite
-  Recharts

Requirements- Node.js and npm

Running the app-  
git clone https://github.com/ashray-bn/Learner_Dashboard.git  
cd Learner_Dashboard  
npm install  
npm run dev

Features:
- Summary of stats such as Total Learners, Average points, Average Assignment % and Highest Score
- A side panel shows points, courses completed, assignment completion %, and current streak for the selected learner.
- A week-by-week line chart (powered by Recharts) showing how the selected learner's score has progressed.

Limitations (Future improvements)
- Static webpage with minimal interactivity
- Performance analytics are limited to weekly score trends provided in the mock dataset
- The application does not support real-time updates or live synchronization across users
- User authentication and role-based access control are not implemented.
